RISC-V Instruction Scheduler

C++

Implemented a RISC-V instruction scheduler based on a simplified LLVM model, covering dependency graph construction, priority calculation, and full List Scheduling workflow. Built an instruction-level DAG to detect three types of data dependencies (RAW/WAR/WAW) optimized for single-pass analysis. Developed List Scheduling algorithm with an Inflight queue, distinguishing release timing of true vs. order dependencies to reduce pipeline stalls.

Python-Based Multi-Line Console Text Editor

Python

Designed and implemented a Vim-style multi-line console text editor supporting over 20 editing commands including insert, delete, cursor movement, line copy/paste, and more. Devised top-down modular architecture with dictionary-based command mapping; integrated stack data structures for multi-level state tracking. Configured regular expressions for word-level cursor jump logic; achieved dual-layer visual display via ANSI escape sequences.
